### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录账号密码登录
2. 注册
3. 人脸识别登录
4. 首页
5. 用户（加好友）
6. 游戏信息
7. 交流论坛
8. 公告信息
9. 留言板
10. 后台管理
11. 个人中心
12. 我的发布
13. 我的收藏



**管理员功能：**
1. 登录账号密码登录
2. 人脸识别登录
3. 个人中心（修改密码、个人信息）
4. 用户管理
5. 游戏分类管理
6. 游戏信息管理
7. 好友信息管理
8. 留言板
9. 交流论坛
10. 系统管理（关于我们、轮播图管理、系统简介、公告信息）
